# 📜 CampusLoop Changelog & Taste Log

A log of significant product updates, UI decisions, and architectural commits.

---

### Recent Updates

- **Modular Codebase & Developer Docs**: Created `src/lib/api.ts` for fetcher/API calls, custom React hooks in `src/hooks/`, and organized documentation in `docs/`.
- **Edit Profile & Category Routes**: Built `/app/profile/edit` for editing profile fields (including gender identification) and `/app/posts/[type]` for clean filtering.
- **One-Tap WhatsApp & Instagram Story Card Sharing**: Instant HTML canvas image generation for 9:16 story sharing.
- **Twitter-Style 1-Tap Reposting & Reaction Animations**: Instant 360° spin repost animations and embedded quoted post cards.
- **Campus Matches (`/app/dating`) Overhaul**: Global radius default, searchable college picker with 1,350+ institutions, drag gestures, and URL filter syncing.
- **Communities & Sub-Hub Fixes**: Fixed server-rendering errors and built clean sub-hub community feed views.
- **Vibe Creator (`/app/stories/new`)**: 9:16 story creation with live canvas preview, gradient selector, sticker badges, and font customization.
